Namespace: SQLiteHelperSingleton

lib/SQLiteHelper~SQLiteHelperSingleton

Singleton accessor for the SQLiteHelper instance. Guarantees that only one database connection exists throughout the application lifecycle. All repositories obtain their database connection through this object.

import SQLiteHelper from '@/lib/SQLiteHelper';
const db = SQLiteHelper.getInstance().getWritableDatabase();
