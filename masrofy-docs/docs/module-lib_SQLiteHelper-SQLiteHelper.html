Class: SQLiteHelper

lib/SQLiteHelper~SQLiteHelper()

Manages the SQLite database connection. On instantiation, it connects to the database file, enables WAL journal mode and foreign key enforcement, and runs the schema initialization script to create tables if they do not exist. This class is not exported directly — access it through the SQLiteHelperSingleton.

new SQLiteHelper()
