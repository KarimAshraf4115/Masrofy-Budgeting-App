Class: User

models/User~User()

Represents the single application user. Manages PIN-based authentication including hashing, verification, lockout enforcement, and attempt counting. The application follows a single-user model — only one User row exists in the database.

verifyPin(inputPin) → {boolean}

Verifies a plain-text PIN against the stored hash. Enforces lockout by returning false immediately if the user is currently locked out. On failure, increments the failed attempt counter and triggers lockout after 5 failures. On success, resets the failed attempt counter.
